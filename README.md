# Rotating Compass Dial

An animated Java visualization of a rotating compass dial using the **Midpoint Circle Algorithm** and **DDA Line Algorithm**.

## Features

- Animated rotating compass with cardinal directions
- Implemented with classic computer graphics algorithms
- Smooth 60 FPS animation with pauses at 45-degree intervals

## Requirements

- Java 8+

## Running

```bash
javac CompassDial.java
java CompassDial
```

## About

CGIP Animation Project - All components contained in a single CompassDial.java file.
